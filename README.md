# nm-s3-samples

## Prerequisites

Serverless

```bash
npm i -g serverless
```

Serverless Compose

```bash
npm i -g @serverless/compose
```

## Deploy

Using serverless compose :

```bash
serverless deploy
```

it will deploy 3 services

- [sample-events](./sample-events/)
- [sample-queries](./sample-queries/)
- [sample-website](./sample-website/)
